# Spring Boot ‚Äî External API Integration

This project demonstrates how to **integrate external APIs** in a Spring Boot application.  
It fetches data from the [JSONPlaceholder API](https://jsonplaceholder.org/posts) and exposes it through your own REST endpoints.

### Features

- **Scalable Project Structure** (Controller, Service, Model, Exception layers)
